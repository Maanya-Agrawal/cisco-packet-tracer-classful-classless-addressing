# Classful and Classless Addressing using Cisco Packet Tracer

## Description

This project demonstrates the implementation of classful and classless IP addressing using Cisco Packet Tracer. A network is designed based on the ASCII value of the first letter of the name, and IP addresses are assigned accordingly. Both addressing schemes are configured and tested to verify proper communication.

## Objectives

* To understand classful and classless IP addressing
* To configure network devices using Cisco Packet Tracer
* To assign IP addresses based on a given addressing scheme
* To verify connectivity using the ping command

## Technologies Used

* Cisco Packet Tracer

## Files Included

* Classful addressing network (.pka)
* Classless addressing network (.pka)
* Report document

## How to Use

1. Open the .pka files using Cisco Packet Tracer
2. Verify IP configurations of all devices
3. Use the ping command to test connectivity

## Conclusion

The experiment demonstrates successful implementation of classful and classless addressing. IP addresses were assigned correctly and network connectivity was verified, confirming proper configuration and subnetting.
